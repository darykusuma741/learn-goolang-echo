# LEARN GOLANG ECHO

A learning repository for Go (Golang) using the Echo framework. This repository contains notes, code examples, and small experiments to understand how to build RESTful APIs, handle routing, middleware, and other web development concepts in Go.

This repository is intended as a documentation of the learning process for Go web development, from basics to more advanced techniques.

## Purpose

- Learn Go web development with Echo
- Understand REST API design and routing
- Experiment with middleware, handlers, and JSON responses
- Serve as a personal reference and portfolio

## Getting Started

### Initialize Project

```bash
go mod init learn-golang-echo
```

### Install Echo

```bash
go get -u github.com/labstack/echo/v4
```

### Running the Project

```bash
go run main.go
```

### Build the Project

```bash
go build
```
